# 考试攻略

> 公务员、事业单位、国考、省考

## 适合谁

* `随缘` 报名考试的佛系同学。
* `时间不多` 经常加班的苦力同学。
* `补习班` 太贵的吃货同学和艰苦同学。
* `无从下手` 被大量无关数据干扰的困惑同学。

## 有什么

* 从 `程序猿为主` 到 `逃离996世界` 的求生指南。
* 从 `网站汇总` 到 `岗位信息` 到 `考试题型` 的考试须知。
* 从 `知识点总结` 到 `快速上手` 的考试技巧。
* 从 `书籍购买` 到 `刷题练级` 的考试实战。
